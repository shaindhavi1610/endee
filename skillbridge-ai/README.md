# SkillBridge AI 🚀

## 📌 Overview
SkillBridge AI is an AI-powered web application that analyzes user skills and suggests the most suitable job roles based on similarity matching.

## 🎯 Objective
To help users identify their best career path by comparing their skills with industry-required skills using simple AI techniques.

## 🚀 Features
- 🔍 Skill analysis
- 🤖 AI-based job role prediction
- 📊 Match percentage display
- 💡 Easy-to-use interface

## 🛠️ Technologies Used
- Python
- Streamlit
- Scikit-learn
- NumPy

## ▶️ How to Run

1. Clone the repository:
   git clone https://github.com/shaindhavi1610/endee

2. Navigate to project folder:
   cd skillbridge-ai

3. Install dependencies:
   pip install -r requirements.txt

4. Run the app:
   streamlit run app.py

## 🧠 How It Works
- User enters skills
- Skills are converted into a simple vector format
- Compared with predefined job skill sets
- Similarity score is calculated
- Best matching role is displayed

## 🧠 Vector Search Concept
This project uses a basic form of vector similarity by comparing overlapping skills between user input and job datasets.
