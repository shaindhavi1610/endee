# SkillBridge AI 🚀

## 📌 Overview
SkillBridge AI is an AI-powered project that analyzes user skills and provides improvement suggestions.

## 🚀 Features
- Skill analysis
- AI recommendations
- Resume/skill matching
- Easy interface

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   streamlit run app.py

## 🧠 Vector Search Concept
This project uses vector similarity to match user skills with job requirements.
